Detailed Description

template<typename T>
class rabbit::Triangle< T >

A triangle class. The following notation is used throughout the triangle related code. The triangle consists of 3 vertices represented as P0, P1 and P2. The corresponding edges are P0P1, P0P2, P1P2 which completes the triangle description.
